# RosterStaff

A rostering system developed in Ionic 3.

## Getting Started
### Prerequisites

- NodeJS (https://nodejs.org/en/)
- Ionic 3

```
npm install -g ionic
```

### Installing Dependencies

Inside the cloned repository, install dependencies using the npm command below

```
npm install
```

## Local Deployment

```
ionic serve
```
